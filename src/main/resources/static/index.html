Nessa versão adicionamos serviços REST para calculos online

Método Requisição Resposta Endpoint
POST { 'first': 10, 'second': 10 } { 'result': 20 } https://www.minhacalculadoraonline.com/calc/sum
POST { 'first': 10, 'second': 10 } { 'result': 0 } https://www.minhacalculadoraonline.com/calc/subtract
POST { 'first': 10, 'second': 10 } { 'result': 100 } https://www.minhacalculadoraonline.com/calc/multiply
POST { 'first': 10, 'second': 10 } { 'result': 1 } https://www.minhacalculadoraonline.com/calc/divide

Também foi adicionada a possibilidade de criar uma area de trabalho e manter um histórico


As seguintes teclas foram adicionadas:

AC : Limpar tela atual

, : Casa Decimal


V1.0.0

Agora você pode fazer suas contas online tranquilamente, online e 100% de graça.

Para entender o funcionamento, basta entender as seguintes teclas:

AC : Limpar tela atual

Limpar : Limpar tela atual e histórico

÷ : Dividir

x : Multiplicar

- : Subtrair

+ : Somar

= : Igual
